# File Storage

---

## Features

### File Storage
- Upload files 
- Download files
- Delete files  

### Authentication
- User registration and login system  
- Secure token-based authentication (JWT)  

### API
- gRPC interface for realtime communication and file operations  
- REST API with Echo framework for additional endpoints (file download, user management) 

---

## Tech Stack

| Component  | Library |
|-----------|---------|
| gRPC                | [grpc-go](https://grpc.io/docs/languages/go/)          |
| REST                | [Echo](https://echo.labstack.com/)                     |
